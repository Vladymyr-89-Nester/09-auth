# NoteHub - Notes Application with Authentication

NoteHub is a modern web application for creating, managing, and viewing notes, built using Next.js. The application includes an authentication system that allows users to register, log in, and manage their notes in a private area.

- **User Authentication**: Registration, login, and logout
- **Note Management**: Create, view, edit, and delete notes
- **Filtering and Search**: Filter notes by categories and search by content
- **Pagination**: Navigate through large lists of notes
- **User Profile**: View and edit profile
- **Modal Windows**: View note details in modal windows
- **Responsive Design**: Works on all devices
